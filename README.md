# 🚒 TNFRS Smart Fire Predictive Intelligence & Command Center
**Selected as a Top 24 Finalist across Tamil Nadu for the TNFSC Innovation Challenge 2026**

### 🌟 Project Overview
This project is an **Integrated Command Center** designed for the Tamil Nadu Fire and Rescue Services (TNFRS). It moves beyond traditional reactive systems by using **Predictive Intelligence** to identify fire hotspots before they occur.

### 🧠 The "Adaptive" Intelligence Engine
Unlike static dashboards, our system features two critical loops:
1. **Continuous Data Ingestion (15s):** Real-time fetches from weather sensors and infrastructure APIs.
2. **Dynamic Model Retraining (120s):** An automated pipeline that "re-learns" environmental risks every 2 minutes.

### 📊 Mathematical Rigor
* **Ensemble Strategy:** Weighted combination of **Random Forest (40%)** and **Gradient Boosting (60%)**.
* **Validated Performance:** * **Accuracy:** 88.78%
  * **$R^2$ Score:** 0.89 (High statistical correlation with historical incident patterns).
* **Dataset:** Trained on a master dataset of **1,223 historical records** from Tamil Nadu.

### 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Machine Learning:** Scikit-Learn, Pandas, NumPy
* **Frontend:** HTML5, CSS3, JavaScript (Dashboard UI)
* **Real-time Integration:** OpenWeather API, Custom JSON Infrastructure Logs

### 🚀 Key Innovations
* **Hyper-Local Dispatch:** Adjusts vehicle recommendations (Fire Bikes vs. Heavy Tankers) based on **Road Width** data in congested areas like T. Nagar.
* **Proactive Pre-positioning:** Recommends moving fire assets to high-risk zones based on rising heat/wind speeds.
